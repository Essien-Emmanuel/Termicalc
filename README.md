## üìñ Usage

### Basic Operations

```sh
> 2 + 2
= 4

> (3 + 2) * 4
= 20

> 2.5 * 3
= 7.5
```

### Keyboard Controls

| Key       | Action                |
|-----------|-----------------------|
| `q`       | Quit application      |
| `Ctrl+C`  | Exit calculator       |
| `Enter`   | Evaluate expression   |

## üõ†Ô∏è Development

### Prerequisites

- Node.js (>= 18.x)
- npm (>= 9.x)

### Local Setup

1. Clone the repository:
   ```sh
   git clone https://github.com/Essien-Emmanuel/Terminal-calculator.git
   cd Terminal-calculator
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

### Scripts

- Start the calculator in development mode:
  ```sh
  npm run calc
  ```

- Build the project:
  ```sh
  npm run build
  ```

- Run tests:
  ```sh
  npm test
  ```
